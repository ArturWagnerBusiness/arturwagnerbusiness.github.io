Why is this website so simple?

I went through many iterations over the years and I think this is the best design. At the time I had been reading about the KISS Principle which stands for "Keep It Simple Stupid". This means that if I need a place to store my portfolio, I don't need a 3d metaverse AI based website with complex API backend. A simple HTML file does the job and does it well. It is also a lot more robust and maintainable!
